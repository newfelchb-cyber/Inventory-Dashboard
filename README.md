# 📦 Inventory Management Dashboard

This project provides a comprehensive solution for tracking stock levels, monitoring product movement, and optimizing inventory control through a fully automated data pipeline.

### 🎯 Objective
To ensure optimal stock availability, minimize excess inventory, and trigger timely reordering through data-driven alerts.

### 📊 Dashboard Features
* **Stock Tracking:** Real-time visibility of current inventory levels.
* **Low-Stock Alerts:** Automated visual indicators for items nearing depletion.
* **Movement Analysis:** Tracking incoming and outgoing stock efficiency.
